🌐 The Web Dashboard

An interactive app for tracking a motorcycle/car flipping business — built from scratch with no frameworks.

This is a single, self-contained web app that runs entirely in the browser — no server, no login, no install. It tracks every vehicle bought and sold, and turns that data into live Inventory and Sales dashboards that update the instant you add or sell a vehicle.

What a recruiter can do with it

Add a vehicle

Click “Add Bike/Car,” enter the details — it appears in inventory immediately.

Mark it sold

Enter a sell price and date; it moves to the Sales view and profit/ROI recalculate.

See the numbers

KPI cards, profit-by-month and profit-by-make charts, and a “most-sold models” ranking.

Track partners

Each vehicle logs partners and auto-calculates each person’s cash share %.

How it was built

  1. Data model first. Each vehicle is a record with buy/sell prices, dates, buyer, and a list of partners. Profit, ROI, and days-to-sell are calculated, never typed.
  2. Pure HTML, CSS & JavaScript. No frameworks — the logic that groups, sorts, and totals the data is hand-written JavaScript.
  3. Live charts. Chart.js renders the profit graphs; they redraw automatically whenever the data changes.
  4. Saved in the browser. Your entries persist locally (localStorage), and there’s CSV/JSON export so the data can move to Excel or a database.
  5. Self-healing. On load it back-fills any missing fields (e.g., partner splits) so the app never shows a broken row.
